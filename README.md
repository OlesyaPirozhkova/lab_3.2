# Задание 8

Напишите функцию для подсчета количества уникальных слов в тексте (без
учета регистра)

<img width="497" height="391" alt="image" src="https://github.com/user-attachments/assets/01e2a0c4-5e51-40ce-8b14-871e3b3d6ce8" />



```
def count_unique_words(text):

    words = text.lower().split()
    
    unique_words = set(words)
    
    return len(unique_words)


print("Введите текст для анализа:")
user_text = input()

if user_text.strip():
    result = count_unique_words(user_text)
    print(f"Количество уникальных слов: {result}")
else:
    print("Вы не ввели текст!")
```

# Задание 10

Напишите функцию, которая находит элементы, уникальные для каждого из
двух списков (симметрическая разность).

<img width="706" height="384" alt="image" src="https://github.com/user-attachments/assets/0f155eaf-d49d-40ed-8831-e59fc0e061c2" />




```
def symmetric_difference(list1, list2):

    return list(set(list1) ^ set(list2))


print("Введите первый список (через пробел):")
list1 = input().split()

print("Введите второй список (через пробел):")
list2 = input().split()


result = symmetric_difference(list1, list2)
print(f"\nУникальные элементы: {result}")
```
